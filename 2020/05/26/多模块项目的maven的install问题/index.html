在springcloud的多模块打包问题中

mvn install 死活打包不了依赖了其他模块的模块，后来百度才发现，是spring-boot-maven-plugin这个玩意，打包的jar包只能运行，不能被依赖

所以要被依赖的jar包请把这个plugin改成下面这样

# java # maven
